# Lab 07

## Co zostało zrealizowane
Rozbudowano projekt o mechanikę walki. Zaimplementowano klasę pocisku z określonym czasem życia (TTL) oraz zarządzanie listami obiektów. Dodano wykrywanie kolizji. Wprowadzono animację eksplozji oraz dodano wczytywanie, odtwarzanie i bezpieczne zwalnianie zasobów z pamięci.

## Uruchomienie
Projekt wymaga języka Python i biblioteki `raylib` (`pyray`). 
Będąc w katalogu `lab_07`, otwórz terminal i wykonaj polecenie:

python main.py
